
This application allows you to create and move rectangular notes ('stickies'), edit text
inside them, change note color, close & reopen your browser, use its undo & redo buttons,
save current notes in a file, and reload this file later.
The application is purely client-sided, with no server part. All data are stored locally.

1. CREATING A NOTE
You create a new note by clicking at any free place of the board. The application cycles 5
color schemes when adding a new note. You can change the color later (see point 4. below).

2. MOVING A NOTE
Note can be simply moved by clicking and moving it to the preferred position. It is not
possible to move a note beyond the edge of the board.

3. EDITING TEXT IN A NOTE
You can edit the text in a note by clicking it and entering text using your keyboard.
Each line of the note serves to keep a specific item. Line wrapping is thus disallowed.
If you attempt to add more characters to a line, or more lines to a note, than fits, the
application refuses to proceed and notifies you by red flash. This holds also for pasting
(Ctrl+V) text into a note.

4. CHANGING NOTE COLOR
While editing a note (see point 3. above), you can press Ctrl+Y for yellow, Ctrl+B for blue,
Ctrl+R for red, Ctrl+G for green or Ctrl+F for fuchsia color of the note.

5. DELETING A NOTE
When you hover over a note, animated grey circle appears at its top right corner. If you
move your mouse cursor into the circle, its color gradually changes to red and black cross (X)
gradually appears. By clicking it, you remove the note immediately.

6. RESIZING THE BOARD (RESIZING BROWSER WINDOW)
Board width is always fits the width of your browser window (thus the board is always fully
visible in the horizontal direction, without any need to scroll it horizontally). As you
resize the board, all notes scale appropriately. Please choose the width of you browser window
to get preferred scale of your notes.
    Height of the board equals 69 % of the width. This rectangular setting usually makes
the whole board visible also in the vertical direction. Otherwise, please reduce the width
of your browser window or scroll vertically.
    You can use the application also from your cell phone. Most users are likely to prefer
the landscape orientation.

7. INFO NOTE
There is one special note (we call it the "info note") that contains clickable text allowing
you to reset the application (see point 8. below), to save all notes (see point 9. below) and
to display this readme file. The info note is already visible when you open the application.
It is not possible to delete the info note, but you can move it to a different position.

8. RESETTING THE APPLICATION (DELETING ALL NOTES)
When you click the "reset" text inside the info note (see point 7. above), the "proceed" text
appears. If you click it, all notes vanish and sound of munched paper plays.

9. SAVING ALL NOTES
When you click "save" inside the info note (see point 7. above), the file representing all notes
(with their text, colors and positions) starts being downloaded. Some browsers may ask you first
whether you wish such upload.

10. LOADING ALL NOTES
After you have saved the file (see point 9. above) to you computer, you can later drag the file
and drop it inside the application window. The application then reverts to the saved state.

11. APPLICATION PERSISTENCE
You can close the browser. When you reopen the application later, you will find it, including
all notes and texts, as you have left it.

12. UNDO & REDO
When you delete, move, focus or recolor your note, or when you reset the application or load
notes from a file, an undo point is automatically created in the application history. This
allows for usual usage of undo & redo buttons of your browser.
